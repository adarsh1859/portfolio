Helmet Detection & Reporting - 2024

This project involves detecting helmet usage among riders using advanced machine learning models and reporting violations.

Technology Used: Python, TensorFlow, PyTorch, OpenCV, Flask

Features:

  • Real-time helmet detection using YOLO-v5
  • Bounding box capturing of rider and vehicle details
  • Integration with Twilio API for SMS alerts

Upon successful authentication, users are directed to dashboard page, which acts as hub for accessing functionalities. The admin can click on choose file and press the upload button

Upon uploading the video input the system detects the rider who are not wearing helmet and apply the bounding box around head region and save the images in the folder with number which can be used for fetching details.

Once the video processing is over then the system collects the images which are store in folder and generates the same number of dynamic reports including its time and date along with the image name.

Upon uploading the video input and after the video processing the detected images are stored in report file which holds the image of rider and number plate. The user can enter the number and there after it displays the information of the rider.

After entering the vehicle registration number and clicking on fetch details the system make use of twilio API which there after sends the SMS notification to the rider.
